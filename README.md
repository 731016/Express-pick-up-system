##### 订单状态

```js
orderStatus
orderStatus_cnName
```

| 订单状态 | 名称     |
| :------: | -------- |
|    10    | 等待接单 |
|    20    | 派送中   |
|    30    | 订单异常 |
|    40    | 订单完成 |
